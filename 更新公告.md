# 🎯 ALInvite 插件更新公告

## 📢 版本 1.0.8 发布！

### 🚀 主要更新内容

#### 🔒 新增功能权限控制系统
- **IP 限制模式**：支持完全禁止同IP绑定或灵活限制功能
- **功能权限控制**：可以单独控制里程碑、返点、礼包功能
- **详细提示消息**：玩家绑定时收到明确的功能状态说明
- **永久绑定状态**：功能权限状态永久记录，不受IP变化影响

#### 📡 增强第三方集成
- **完整的 ALInviteAPI 接口**：提供16个API方法
- **异步处理**：所有API方法支持CompletableFuture
- **丰富的集成场景**：支持邀请码管理、邀请统计、返点处理等
- **详细的API文档**：包含使用示例和参数说明

#### 🛠️ 配置优化
- **简化IP限制配置**：更清晰的配置结构
- **详细的配置注释**：每个配置项都有详细说明
- **默认配置优化**：更合理的默认设置

#### 📚 文档更新
- **WIKI文档**：完全重写，包含所有功能详细说明
- **API文档**：完整的ALInviteAPI接口文档
- **配置说明**：详细的配置项说明

### 🔧 修复的问题
- **绑定逻辑**：修复了同IP绑定的功能权限控制
- **第三方集成**：移除了不稳定的第三方支付插件集成
- **命令调用**：优化了命令执行逻辑
- **语言文件**：修复了部分消息文本

### 🎯 功能权限控制配置示例

```yaml
# IP 限制配置
ip_restriction:
  enabled: false                 # 是否启用同IP绑定限制
  # 如果 enabled=true：完全禁止同IP绑定（下面的配置无效）
  # 如果 enabled=false：允许同IP绑定，但根据 flexible_mode 限制功能
  flexible_mode:
    milestone: false             # 同IP绑定：里程碑功能禁用
    rebate: true                 # 同IP绑定：返点功能启用
    gift: false                  # 同IP绑定：礼包功能禁用
```

### 📦 安装方法
1. 下载 `ALInvite-1.0.8-shaded.jar`
2. 放入服务器 `plugins` 文件夹
3. 重启服务器
4. 根据需要修改 `config.yml` 配置

### 🔄 配置迁移
- 插件会自动迁移旧配置
- 新配置项会自动添加到配置文件中
- 建议重新检查配置文件确保设置正确
